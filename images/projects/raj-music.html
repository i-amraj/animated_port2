Project Overview

Raj Music Player is a modern web-based music streaming application that offers a seamless listening experience. It features a beautiful user interface with audio visualizations, playlist management, and smart recommendations. The player supports various audio formats and includes features like equalizer settings, lyrics display, and offline mode support.

Key Features

Audio Streaming

High-quality audio streaming with support for multiple formats and bitrates.

Playlist Management

Create, edit, and organize playlists with drag-and-drop functionality.

Audio Visualizer

Real-time audio visualization with multiple styles and customization options.

Equalizer

Built-in equalizer with presets and custom settings for personalized sound.

Favorites & Library

Save favorite tracks and build your personal music library.

Responsive Design

Fully responsive interface that works beautifully on all devices.
